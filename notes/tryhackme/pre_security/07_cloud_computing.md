# Cloud Computing

## What is cloud computing?
Cloud computing to korzystanie z serwerów przez internet zamiast na własnym sprzęcie.

👉 zamiast kupować własne serwery:
- korzystasz z zasobów online  
- płacisz tylko za użycie  

---

## How cloud evolved

1. **Physical servers**
   - firma kupuje własny sprzęt  
   - musi go utrzymywać  
   - skalowanie = kupowanie nowych serwerów  

2. **Virtualization**
   - jeden serwer → wiele maszyn wirtualnych  
   - lepsze wykorzystanie zasobów  
   - tańsze rozwiązanie  

3. **Cloud**
   - zasoby dostępne przez internet  
   - dostęp z każdego miejsca  
   - łatwe skalowanie  

---

## Cloud Features

- **Scalability** → zwiększanie/zmniejszanie zasobów  
- **Pay-as-you-go** → płacisz tylko za użycie  
- **High availability** → działa nawet przy awarii części systemu  
- **Global access** → dostęp z całego świata  
- **Self-service** → szybkie tworzenie zasobów  
- **Security** → dostawca zabezpiecza infrastrukturę  

---

## Types of Cloud

- **Public Cloud**  
  dostęp przez internet, używany przez wiele firm  

- **Private Cloud**  
  dedykowany dla jednej organizacji  

- **Hybrid Cloud**  
  połączenie publicznej i prywatnej  

---

## Cloud Service Models

### IaaS (Infrastructure as a Service)
- dostajesz serwer, sieć, zasoby  
- sam konfigurujesz system (np. Linux)  

---

### PaaS (Platform as a Service)
- dostajesz gotowe środowisko  
- wrzucasz tylko aplikację  

---

### SaaS (Software as a Service)
- używasz gotowej aplikacji przez internet  
- np. Google Docs  

---

## Example: EC2

EC2 to usługa chmurowa (Amazon), która pozwala:
- tworzyć wirtualne maszyny  
- zmieniać ich rozmiar  
- uruchamiać je w dowolnym momencie
